# Campus Cash Calc - Student Finance Tracker

A responsive, accessible vanilla HTML/CSS/JS finance tracker for students.

## Features
- Track income and expenses with categories
- Set a monthly budget cap with visual progress bar
- 7-day spending trend chart
- Regex-powered live search
- Sort and filter records
- Import / Export JSON
- Full keyboard and screen reader accessibility

## Tech Stack
Vanilla HTML5, CSS (mobile-first), JavaScript (ES5 compatible).
No frameworks. No build tools.

## Running Locally
Open `index.html` in any modern browser. No server needed.

## Tests
Open `tests.html` to run the automated validator test suite (R1-R5).
